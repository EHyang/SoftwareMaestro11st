## 서버 구동 방법


1. Node.js 설치

   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
   . ~/.nvm/nvm.sh
   nvm install node
   ```

2. Package 설치

   ```sh
   npm i -g
   ```

3. 실행

   ```sh
   npm start
   ```
